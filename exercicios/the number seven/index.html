Negrito e Italico

aqui irá ter uma palavra em negrito volta vida, ainda te amo e aqui provavelmente é uma palavra em italico

B (sem semantica) / STRONG (com semantica) é para palavras em negrito como essa "cruzeiro cabuloso" e I (sem semantica)/ EM (com semantica) para palavras em italico como essa cruzeiro maior de minas

Texto marcado

podemos marcar um texto usando a tag MARK usando o metodo de ctrl + shift + p
sempre use o comando style quando for marcar dentro do < >, e a configuração de cor só ficará dentro desse mark

para fazer a cor do mark virar padrão você deverá ir no head digitar style + mark abrir chaves e digitar o codigo background color , um exemplo disso você verá agora o cruzeiro é o maior time de todos os tempos

Texto deletado

podemos usar a tag del para riscar um texto como o exemplo seguir Volta vida
